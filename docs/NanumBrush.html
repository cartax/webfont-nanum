NanumBrush

Usage


HTML

<link rel="stylesheet" href="file/path/font.css" />

CDN

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/cartax/webfont-nanum/dist/NanumBrush/subset/{kr, kr-hanja}/font.css" />

CSS

body { 
    font-family: 'NanumBrush', serif; 
    font-weight: 400;
}
